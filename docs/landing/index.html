Project Overview

A comprehensive AWS Lambda-based and Microservices API designed to support vocational guidance and career exploration for students in Guatemala and Latin America. Built as a graduation project at Universidad del Valle de Guatemala.

Educational Mission

Designed to help students discover their career paths through data-driven insights and interactive content.

  • Career exploration tools
  • Interactive forums and discussions
  • Personalized recommendations
  • Alumni success stories

Serverless Architecture

Built on AWS Lambda with individual functions for each feature, ensuring scalability and cost-effectiveness.

  • Independent Lambda functions
  • Auto-scaling capabilities
  • Pay-per-use pricing
  • High availability

Security First

Implements multiple layers of security including data encryption, JWT authentication, and traffic encryption.

  • AES-256 encryption for data at rest
  • TLS 1.3 style traffic encryption
  • JWT-based authentication
  • Role-based access control

Why Mirai 未来?

[ mi-rai ] • Japanese: "Future"

Mirai (未来) means "future" in Japanese. This name embodies our core mission: to shape and empower the future of vocational orientation education across Guatemala and Latin America.

Building Tomorrow's Professionals

We believe every student deserves access to intelligent, personalized guidance that helps them discover their true calling and potential career paths.

Innovation in Education

By combining AI-powered recommendations, comprehensive assessments, and modern technology, we're revolutionizing how students explore and choose their careers.

Impact Across Latin America

Starting in Guatemala, our vision extends to empowering students throughout Latin America with tools to make informed decisions about their future.

"The future belongs to those who prepare for it today. Mirai is our contribution to ensuring that every student has the opportunity to build their own future with confidence, knowledge, and the right guidance."

Security Features

Multi-layered security implementation for data protection and user privacy

Web Application Firewall (WAF)

Advanced AWS WAF protection against multiple attack vectors and malicious traffic patterns.

  • DDoS attack protection and mitigation
  • SQL injection prevention
  • Cross-site scripting (XSS) protection
  • Rate limiting and throttling
  • Geographic restrictions and filtering
  • Bot traffic detection and blocking
  • Custom security rules and policies

CloudFront Security

Global CDN with built-in security features for content protection and secure delivery.

  • HTTPS enforcement and SSL/TLS termination
  • Origin access control (OAC)
  • Signed URLs and cookies
  • Geographic restrictions
  • Edge caching security
  • DDoS protection at edge locations

VPC Private Cloud Security

Comprehensive network isolation and security within AWS Virtual Private Cloud for maximum data protection.

  • Private subnet isolation for all services
  • Security groups with least privilege access
  • NAT Gateway for secure outbound internet access
  • Route tables for controlled network traffic
  • VPC Link for secure microservice communication
  • Network ACLs for additional layer protection
  • IPv4 private addressing for internal services

Data Encryption

Traffic (HTTPS) and rest (AES-256) encryption for data protection. And additional encryption for Personal Identifiable Information.

JWT Authentication

Secure token-based authentication with public key verification and role-based access control.

Role-Based Access

Granular permissions system with admin, teacher, director, and student roles.
