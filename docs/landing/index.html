Mirai Server

Serverless Backend for Vocational Orientation Platform

A comprehensive AWS Lambda-based API designed to support vocational guidance and career exploration for students in Guatemala and Latin America. Built as a graduation project at Universidad del Valle de Guatemala.

Explore API View Architecture

Project Overview

A graduation work project aimed at contributing to vocational orientation in Guatemala and Latin America

Educational Mission

Designed to help students discover their career paths through data-driven insights and interactive content.

  • Career exploration tools
  • Interactive forums and discussions
  • Personalized recommendations
  • Alumni success stories

Serverless Architecture

Built on AWS Lambda with individual functions for each feature, ensuring scalability and cost-effectiveness.

  • Independent Lambda functions
  • Auto-scaling capabilities
  • Pay-per-use pricing
  • High availability

Security First

Implements multiple layers of security including data encryption, JWT authentication, and traffic encryption.

  • AES-256 encryption for data at rest
  • TLS 1.3 style traffic encryption
  • JWT-based authentication
  • Role-based access control

Security Features

Multi-layered security implementation for data protection and user privacy

Data at Rest Encryption

AES-256-CBC encryption for all personal identifiable information (PII) stored in the database.

Traffic Encryption

AES-256-GCM encryption for data in transit, providing TLS 1.3 level security.

JWT Authentication

Secure token-based authentication with public key verification and role-based access control.

Role-Based Access

Granular permissions system with admin, teacher, director, and student roles.

API Endpoints

RESTful API endpoints for all platform functionality

Authentication

POST /auth/register
Register a new user account
DELETE /auth/delete
Delete user account

Content & Exploration

GET /explore/feed
Get paginated content feed with filtering
POST /explore/cards/newCard
Create new content card
GET /explore/cards/getCard
Retrieve card by ID
POST /explore/interactions/newInteraction
Record user interaction

Forums

GET /forums/getForums
Get all forum posts
GET /forums/getForum/{id}
Get specific forum post
POST /forums/newForum
Create new forum post
POST /forums/comments/newComment
Add comment to forum

Careers

GET /careers/getCareers
Get all careers with filtering
GET /careers/getCareer/{id}
Get detailed career information
PUT /careers/editCareerInsights
Update career insights

User Management

GET /users/getUsers
Get all users (admin only)
GET /users/getUser/{id}
Get user profile
PUT /users/editUser/{id}
Update user profile
POST /users/saved/saveItem
Save item to user collection

Analytics

GET /analytics/getAnalytics
Get platform analytics
GET /quiz/results/getResults
Get quiz results
